## clone and run
```shell
  git clone xxx
  cd ShangPinHuicom
  npm run serve
```
## 技术栈
- vue2全家桶
- mock数据 模拟floor楼层数据、banner轮播图数据
- nprogress 请求api加载动画
- vue-lazyload 实现图片懒加载
- vee-validate@2 对登陆和注册进行表单合法性校验
- element-ui messageBox-支付弹窗
- QRcode 生成支付二维码
